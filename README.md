## 📁 Como baixar o projeto

```bash

    # Clonar o repositório
    $ git clone https://github.com/lrdplopes/breakingbadAPI

    #Acessar o repositório
    $ cd breakingbadAPI

    #Instalar suas dependências
    $yarn install

    #Iniciar o projeto
    $yarn start

```
